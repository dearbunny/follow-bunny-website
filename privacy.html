Follow Bunny 隱私政策

歡迎使用 Follow Bunny（以下簡稱「本應用程式」）。您的隱私對我們來說非常重要，因此我們制定了本隱私政策，以解釋我們如何收集、使用和保護您的個人資訊。

1. 我們收集的資訊

當您使用本應用程式時，我們可能會收集以下資訊：

  • Instagram API 提供的公開資訊（如使用者名稱、個人資料圖片）。
  • 您的追蹤名單和粉絲名單（僅用於 APP 內部分析）。

2. 資訊的使用方式

我們收集的資訊將用於以下目的：

  • 提供 Instagram 追蹤分析服務。
  • 改善應用程式的使用體驗。
  • 確保符合 Instagram API 使用規範。

3. 資訊共享與保護

我們不會向第三方出售、交換或公開您的個人資訊。您的數據將受到適當的保護，確保不會被未授權存取。

4. 用戶權利

您可以隨時撤回對本應用程式的 Instagram API 授權，並刪除您的帳戶數據。

5. 政策更新

我們可能會不定期更新本隱私政策，請定期查閱以確保了解最新資訊。
